The option to create your API key can be found within the settings menu.

<Frame caption="Project settings menu icon, 12/2025">
  <img src="/images/accounts/project-settings.png" alt="project setting" />
</Frame>

Click on `Create an API key`.

<Frame caption="Create an API key button, 12/2025">
  <img
    src="/images/run/create-API-key.png"
    alt="Link to create an API key in the Octomind app"
  />
</Frame>

You then will be navigated to our auth provider where you can create a new organization-level API key.

Add the API key to your CI provider as a secret to run test reports from the CI.
