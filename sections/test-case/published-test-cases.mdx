# Overview

In the overview layout, you can observe: **date** of your test case creation, **name** of the test case, **URL** you provided, and its **last run.**

The actions you can perform include:
- **transform published test case to archived** by clicking on the delete icon located in the top right corner
- pressing **last run**, which will redirect you to a new page where you can view the test report in Playwright Trace Viewer
