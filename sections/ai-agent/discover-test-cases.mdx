# Draft state

New draft state that will be presented here is
- **Needs approval**

It is draft state that you can either **approve** or **discard.** If you find the provided results satisfactory, you can
press `approve` button and keep it as a test case. Alternatively, if the situation is unfavorable, you can press the `discard` button.
