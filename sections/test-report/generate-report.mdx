In order to generate a report, you must choose one of the following four options:
- **in app**
- **in app via terminal**
- **via CI**
- **locally**

# in app

Upon selecting this option your report generation will start immediately and results will be visible within the application
once the process is completed.

# in app via terminal

Following the selection of this option, pop up will appear prompting you to **copy** a command and **execute it within the terminal.**
After execution, navigate to the application's **test reports** section and your results will be visible.

Additionally, you will need to generate an **API key** and incorporate it into the process.

To **create a new API key**, navigate to the settings in the top right corner and click on the `Create API key` button.
Upon redirection, click on the `+ New API key` button. Your API key is ready for use!

# locally

You can generate your report **locally** and view the test results on your local machine, where you can also debug it, if needed.

Simply select the `locally` option, then copy the command and execute it in your terminal.
